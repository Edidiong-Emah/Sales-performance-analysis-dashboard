Sales Performance Analysis Dashboard
Project Overview

This project focuses on analyzing sales performance data using Microsoft Excel to identify trends, outliers, revenue patterns, and key business insights.

The project involved:

Descriptive statistics
Trend analysis
Outlier detection
Revenue analysis
Dashboard visualization
Business recommendations

The objective was to transform raw sales data into meaningful insights that support data-driven decision-making.

Key Insights
Certain products generated significantly higher revenue than others.
Gross revenue differed from actual realized revenue due to cancelled and returned orders.
High-value transactions influenced overall sales averages.
Some sales periods showed stronger revenue trends than others.
Order cancellations and returns contributed to revenue loss.
Recommendations
Focus on reducing cancellations and product returns.
Increase investment in high-performing products.
Monitor outlier transactions for premium customer opportunities.
Use trend analysis for forecasting and inventory planning.
Continue leveraging dashboards for data-driven decision-making.
